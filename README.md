# Fitness Tracker App

## Description
Fitness Tracker is a simple and user-friendly application to help users monitor their workouts, diet, and overall health progress. The app is designed for anyone who wants to maintain a healthy lifestyle, track calories, and achieve fitness goals based on the user input.

---

## Features
- Track daily workouts with exercise name, duration, and calories burned
- Monitor daily diet with meals, calories, and nutrients (protein, carbs, fats)
- View weekly and monthly progress charts based on user input
- Set personalized fitness goals
- Simple and intuitive interface

---

## Tech Stack
- **Frontend:** thymeleaf
- **Backend:** Java / Spring Boot
- **Database:** postgreSql
- **Libraries & Tools:**  Gradle
